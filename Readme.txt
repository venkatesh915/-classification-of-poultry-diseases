# Flask Web App for Poultry Disease Classification

## How to Run
1. Place your trained model as `healthy_vs_rotten.h5` in the root directory.
2. Install dependencies: `pip install flask tensorflow`
3. Run the app: `python app.py`
4. Open your browser and go to: http://127.0.0.1:5000

Upload a poultry image to get disease classification prediction.
